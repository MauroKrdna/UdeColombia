📌 INTRODUCCIÓN
---------------------------------------
El *Student Performance Dataset* fue diseñado para analizar los factores que influyen en
el rendimiento académico de los estudiantes. Contiene 10,000 registros sintéticos que combinan
datos de estudio, hábitos y actividades extracurriculares.

Este dataset es ideal para investigaciones educativas, proyectos de ciencia de datos
y entrenamiento de modelos predictivos.

📄 DESCRIPCIÓN DEL DATASET
---------------------------------------
Cada registro incluye las siguientes variables:

- Hours Studied: Número total de horas de estudio por estudiante
- Previous Scores: Puntajes obtenidos en pruebas anteriores
- Extracurricular Activities: Participación en actividades extracurriculares (Yes / No)
- Sleep Hours: Promedio de horas de sueño diario
- Sample Question Papers Practiced: Cantidad de exámenes de práctica resueltos

📌 VARIABLE OBJETIVO
---------------------------------------
- Performance Index: Índice de desempeño académico (valor entero entre 10 y 100,
  donde valores más altos indican mejor rendimiento).

⚠️ IMPORTANTE
---------------------------------------
- El dataset es completamente **sintético**, creado con fines ilustrativos.
- Las relaciones entre variables no necesariamente reflejan datos reales.

💡 POSIBLES APLICACIONES EN MACHINE LEARNING
---------------------------------------

🔹 REGRESIÓN
- Predecir el 'Performance Index' usando modelos como regresión lineal, regresión por bosques aleatorios o redes neuronales.

🔹 CLASIFICACIÓN
- Agrupar a los estudiantes en categorías de rendimiento (ej. bajo, medio, alto) a partir del 'Performance Index'.

🔹 REGRESIÓN LOGÍSTICA
- Predecir la probabilidad de que un estudiante supere cierto umbral de rendimiento (ej. Performance Index ≥ 75).

🔹 CLUSTERING
- Detectar patrones o segmentos de estudiantes según hábitos de estudio y resultados académicos.

📝 LICENCIA
---------------------------------------
Este dataset es de uso libre para compartir, analizar o modificar.
